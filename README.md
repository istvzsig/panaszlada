# Panaszláda

**Live demo:**  
👉 https://panaszlada.onrender.com

---

## 📌 Mi ez?

A **Panaszláda** egy nyílt forráskódú, közösségi alapú állampolgári bejelentő platform.

A cél, hogy az állampolgárok egyszerűen, akár anonim módon jelezhessenek közérdekű problémákat (pl. kátyúk, közvilágítás, szemét, hibák), amelyek egy interaktív térképen jelennek meg és követhetők.

A projekt hosszú távú célja az állampolgári részvétel erősítése és a közszolgáltatások átláthatóságának növelése.

---

## ⚙️ Tech stack

- Backend: Go
- Database: PostgreSQL (Supabase)
- Frontend: Vanilla JavaScript
- Map: MapLibre GL JS
- Deployment: Render

---

## ✨ Funkciók (MVP)

### 📍 Bejelentések

- Anonim bejelentés létrehozása
- Cím + leírás
- Helyszín megjelölése térképen
- Kategorizálás
- Egyedi tracking code

### 🗺️ Térképes megjelenítés

- Interaktív térkép
- Marker alapú megjelenítés
- Popup információk
- Bejelentések vizuális böngészése

### 🧑‍💼 Admin (MVP)

- Bejelentések listázása
- Státusz kezelés
- Törlés / moderáció

---

## 🚧 Fejlesztés alatt

A következő funkciók tervezettek:

- Frontend UI/UX fejlesztés
- Marker clustering (sok report esetén)
- Kategória rendszer bővítése
- Képfeltöltés
- Felhasználói autentikáció
- Moderációs workflow
- Értesítések
- Statisztikák / dashboard

---

## 🤝 Közreműködés

Jelenleg **frontend fejlesztőt keresek**, aki segítene:

- UI/UX finomításban
- térkép interakciók fejlesztésében

- általános frontend polishing-ban

Ez egy működő end-to-end MVP, most a fókusz a felhasználói élmény fejlesztése.

Ha érdekel a projekt, nyugodtan:

- írj üzenetet
- nyiss issue-t
- vagy csatlakozz később contributor-ként

## 🎯 Küldetés

A Panaszláda célja egy olyan digitális platform létrehozása, amely segíti az állampolgárok és közintézmények közötti kommunikációt, miközben tiszteletben tartja az anonimitást és az adatvédelmet.
